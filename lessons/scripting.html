Javascript

What is Javascript?

Javascript allows you to make your HTML documents dynamic by adding scripts to them.

What's a function?

A function is a self-contained chunk of code created to do a single, specific task.

What's a method?

Think of them as verbs. Methods are the behavior assigned to objects.
