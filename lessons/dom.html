The DOM

What is the DOM?

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It provides a structured representation of the document and it defines a way that the structure can be accessed from programs so that they can change the document structure, style and content. The DOM provides a representation of the document as a structured group of nodes and objects that have properties and methods. Essentially, it connects web pages to scripts or programming language.
The document object model.

It is platform and language neutral. HTML is a very common markup language used with the DOM specifically to create web pages.

We talk more about the DOM because that's where our HTML lives. But the DOM is just an object itself contained within the Browser Object Model (BOM). This is moe important to understanding how javascript is used in webpages and we'll talk more about it later.
