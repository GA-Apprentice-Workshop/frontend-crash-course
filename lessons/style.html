CSS

What is CSS?

CSS stands for cascading style sheets. They’re hierarchical and styles are displayed in a cascading fashion. The order and placement of your styles matter. HTML elements can have attributes and those are described with CSS.

What is the Box Model?

The box model is a convention for designing a simple site layout using block items to contain all page elements. It’s common practice to use a lot of

tags for this. The important thing is that the page is a box of perfectly aligned boxes.
